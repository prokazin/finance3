# Telegram Mini App: Личный Финансовый Учёт

## Описание
Приложение для Telegram Web Apps. Позволяет:
- Добавлять доходы, расходы, долги
- Сохранять данные локально
- Быстрые кнопки +100/+500
- Графики с категориями и иконками
- Рассчитывать остаток зарплаты

## Установка
1. Зарегистрируйте бота в BotFather.
2. Вставьте токен бота в `.env` (если используете серверный код).
3. Загрузите проект на Vercel:
```bash
vercel deploy
